## About

Modern Green Company is a company focused on providing eco-friendly products and services. This project is a responsive website built with React and React-Bootstrap, featuring smooth navigation between pages.

---

## Features

- Fully responsive design using Bootstrap.
- React Router navigation between pages.
- FontAwesome icons for modern UI.
- Clean and simple UI/UX.
- Header and Footer visible on all pages.

---

## Pages

1. **Home** – Main landing page with overview of the company.  
2. **Services** – Displays all the services offered.  
3. **Projects** – Shows ongoing and completed projects.  
4. **Products** – Lists all products provided by the company.  
5. **Articles** – Contains news, tips, and blog posts.  
6. **Contact Us** – Form and contact information for users.
